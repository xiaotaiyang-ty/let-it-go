# 🌿 不内耗 (Let It Go)

> 换个角度看问题，事情往往没那么糟糕。

一个帮助你缓解焦虑和内耗的小工具。当你陷入负面情绪时，它会陪你聊聊，帮你换个视角看问题。

**🔗 在线体验：** [let-it-go-gilt.vercel.app](https://let-it-go-gilt.vercel.app)

---

## ✨ 功能特色

### 🔄 换位思考
当你和别人产生矛盾、感到委屈时，AI 会 100% 代入对方视角，用第一人称还原 ta 的真实想法——不美化、不洗白，帮你理解「ta 为什么这样做」。

### ☀️ 停止灾难化
当你把事情往最坏的方向想时，AI 会帮你分析：这件事真的会那么糟吗？最坏情况发生的概率有多大？即使发生了又能怎样？

### 💬 日常陪伴
不想用特定功能？也可以随便聊聊。AI 不会说教，只会倾听和陪伴。
